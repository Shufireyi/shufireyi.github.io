# 第一步：安装hexo

node和npm此处不表。

 npm i -g hexo-cli

安装成功后：

hexo i myblog  // 生成博客文件夹
cd myblog      // 进入目录,其中官方给出了一个demo
hexo g         // 生成js、css、html等静态文件
hexo s         // 启动一个本地服务器，预览页面

本地使用浏览器，输入地址localhost:4000,进行预览。 {% asset_img demohome.jpg %}

# 第二步：需要注册一个github账号

# 1.打开最大的同性交友网站github,进行注册。

{% asset_img githubHome.jpg %}

# 2.新建一个仓库，仓库名要以github.io结尾

{% asset_img newR.jpg %} 因为我之前已经新建过同名仓库，此处报同名错误。

# 第三步：将hexo生成的博客内容上传至github

# 1.编辑 _config.yml 文件, repo 设置成自己的仓库地址。

{% asset_img config.jpg %}

# 2.开始同步

npm install hexo-deployer-git --save    // 上传需要的插件
hexo d   // 部署

# 3.查看效果

{% asset_img result.jpg %}

# 第四步：新建博客

hexo n "blog name" // 在 `source/_posts/` 文件夹下会生成对应的*.md文件

现在编辑你的博客吧！采用markdown语法～

接着：

hexo clean // 清除之前生成的文件
hexo g // 重新生成
hexo d // 部署到github

大功告成！快去看看自己的第一篇博客～ 如果不喜欢这个样式，hexo有很多主题供选择。可以慢慢挑选！
